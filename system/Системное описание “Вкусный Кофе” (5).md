# База знаний «Вкусный Кофе»

Этот репозиторий содержит системные описания, регламенты, материалы для обучения и франшизы сети кофеен **«Вкусный Кофе»**.

## Структура репозитория

- `system/` — системные описания сети и архитектура системы  
  - (здесь лежит основной файл с системным описанием сети кофеен — укажи ссылку на свой файл, например  
    `[Системное описание сети](system/Системное описание "Вкусный Кофе" (5).md)` )

- `training/` — обучение и стандарты для команды  
  - [Системное описание для бариста](training/barista-system-description.md)

- `franchise/` — материалы для франчайзи и партнёров  
  - [Презентация франшизы и подхода](franchise/presentation.md)

- `barista-class/` — проект обучения «Бариста Класс»  
  - [Описание проекта](barista-class/README.md)

---

## Как пользоваться этой базой

- Руководству и архитектору системы — раздел `system/`  
- Франчайзи и партнёрам — раздел `franchise/`  
- Обучение сотрудников и бариста — раздел `training/`  
- Отдельный проект обучения бариста — раздел `barista-class/`

При добавлении новых документов:
- клади их в подходящую папку;
- при необходимости добавляй ссылки сюда в раздел «Структура репозитория».
