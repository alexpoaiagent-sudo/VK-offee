# База знаний сети «Вкусный Кофе»

Эта база собирает операционные документы, инструкции и визуальные материалы сети кофеен «Вкусный Кофе». Все материалы приведены к латинице и формату kebab-case, чтобы их легко подключать к автоматизации и обновлять централизованно.

**Для кого:**
- Бариста — быстрые чек-листы, меню и подсказки по выкладке.
- Официант/раннер — должностные инструкции, вопросы по кассе и контроль смены.
- Управляющий — финансы, поставщики, контроль качества и брендовые материалы.
- Операционный менеджер — дорожные карты, светофор качества, маршруты обновления базы.

## Как добавить новый документ
1. Помести файл в подходящий раздел (system/training/ops/vendors/finance/assets) и придерживайся латиницы + kebab-case.
2. Если нужен отдельный раздел в /ru/, добавь ссылку в соответствующий README (по ролям, процессам или разделам).
3. Для новых процессов используй [шаблон документа](templates/doc-template.md) и пометь статус в светофоре.
4. Обнови `knowledge-inventory.md`: добавь строку в реестр и, при необходимости, статус светофора.
5. Закоммить изменения с понятным сообщением и отправь pull request на ревью.

## Структура репозитория
- `ru/` — русская витрина и навигация по ролям, процессам и разделам.
- `system/` — глоссарий, ценности и карточки контроля качества.
- `training/` — учебные материалы и конспекты.
- `ops/` — операционные инструкции, роли и чек-листы.
- `vendors/` — контакты поставщиков и прайсы.
- `finance/` — расчеты себестоимости и калькуляции.
- `assets/` — меню, фото-подсказки и визуальные материалы.
- `docs/import/` — история переноса из внешних источников.
