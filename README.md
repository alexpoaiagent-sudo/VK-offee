# База знаний «Вкусный Кофе»

Этот репозиторий содержит системные описания, регламенты, обучающие материалы и презентации сети кофеен «Вкусный Кофе».

## Структура репозитория

- `system/` — системные описания сети и архитектура системы  
  - [Системное описание сети](system/Системное описание «Вкусный Кофе» .md)

- `training/` — обучение и стандарты для команды  
  - [Системное описание для бариста](training/barista-system-description.md)

- `franchise/` — материалы для франчайзи и партнёров  
  - [Презентация франшизы и подхода](franchise/presentation.md)

- `barista-class/` — проект обучения «Бариста класс»  
  - (сюда позже добавим подробное описание проекта и оглавление)

---

## Как пользоваться этой базой

- Руководителю / архитектору системы — раздел **`system/`**  
- Команде и бариста — раздел **`training/`**  
- Партнёрам и франчайзи — раздел **`franchise/`**  
- Отдельное направление обучения бариста — **`barista-class/`**

При добавлении новых документов:
- размещай их в подходящей папке;
- при необходимости добавляй ссылки на них сюда, в раздел «Структура репозитория».
